# Scale a float by 3.0f

The literal `3.0f` is loaded from the small-data area with **`lfs`**, then a
single-precision **`fmuls`** does the work. Keeping everything `f32` avoids a
`frsp` rounding step.

## Your task
Write `scale` so it compiles to the `fmuls` above.

<!-- starter -->
```c
f32 scale(f32 x) {
    return 0.0f;
}
```

<!-- solution -->
```c
f32 scale(f32 x) {
    return x * 3.0f;
}
```
