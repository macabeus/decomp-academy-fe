# Logical shift left by 1

`x << 1` is the extended mnemonic **`slwi r3, r3, 1`**, itself a special
`rlwinm` (rotate-left-1, mask the low 1 bits to zero).

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 1;
}
```
