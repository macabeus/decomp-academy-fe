# Unsigned divide by 256

Dividing an **unsigned** value by the power of two 256 discards the low 8
bits — a single **`srwi r3, r3, 8`**. No rounding fix-up is needed (that's
only for *signed* division by a power of two).

## Your task
Write `udiv` on a `u32` so it compiles to the `srwi` above.

<!-- starter -->
```c
u32 udiv(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 udiv(u32 x) {
    return x / 256;
}
```
