# Multiply by 2

The optimizer never emits a register `mullw` for a constant. Because 2 is a power of two, `x * 2` is just a left shift (`slwi r3, r3, 1`).

## Your task
Write `scale` to match the target.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 2;
}
```
