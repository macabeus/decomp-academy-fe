# Logical shift left by 5

`x << 5` is the extended mnemonic **`slwi r3, r3, 5`**, itself a special
`rlwinm` (rotate-left-5, mask the low 5 bits to zero).

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 5;
}
```
