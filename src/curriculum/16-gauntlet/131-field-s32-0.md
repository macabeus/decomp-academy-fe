# Load a `s32` at byte offset 0

A typed field access compiles to a single load with the field's displacement.
A `s32` field reads with **`lwz`** at displacement 0. The *type* picks the load opcode; the *offset* picks the displacement.

```c
typedef struct { s32 field; } S;
```

## Your task
Write `read` to reproduce the assembly above.

<!-- starter -->
```c
s32 read(S* s) {
    return 0;
}
```

<!-- solution -->
```c
s32 read(S* s) {
    return s->field;
}
```

