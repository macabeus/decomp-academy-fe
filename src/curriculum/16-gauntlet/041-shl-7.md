# Logical shift left by 7

`x << 7` is the extended mnemonic **`slwi r3, r3, 7`**, itself a special
`rlwinm` (rotate-left-7, mask the low 7 bits to zero).

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 7;
}
```
