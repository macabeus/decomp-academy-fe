# Logical shift left by 10

`x << 10` is the extended mnemonic **`slwi r3, r3, 10`**, itself a special
`rlwinm` (rotate-left-10, mask the low 10 bits to zero).

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 10;
}
```
