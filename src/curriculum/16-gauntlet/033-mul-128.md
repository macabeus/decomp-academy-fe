# Multiply by 128

The optimizer never emits a register `mullw` for a constant. Because 128 is a power of two, `x * 128` is just a left shift (`slwi r3, r3, 7`).

## Your task
Write `scale` to match the target.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 128;
}
```
