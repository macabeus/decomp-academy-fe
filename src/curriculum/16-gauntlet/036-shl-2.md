# Logical shift left by 2

`x << 2` is the extended mnemonic **`slwi r3, r3, 2`**, itself a special
`rlwinm` (rotate-left-2, mask the low 2 bits to zero).

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 2;
}
```
