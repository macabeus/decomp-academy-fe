# Load a `u8` at byte offset 24

A typed field access compiles to a single load with the field's displacement.
A `u8` field reads with **`lbz`** at displacement 24. The *type* picks the load opcode; the *offset* picks the displacement.

```c
typedef struct { u8 _pad[24]; u8 field; } S;
```

## Your task
Write `read` to reproduce the assembly above.

<!-- starter -->
```c
u8 read(S* s) {
    return 0;
}
```

<!-- solution -->
```c
u8 read(S* s) {
    return s->field;
}
```
