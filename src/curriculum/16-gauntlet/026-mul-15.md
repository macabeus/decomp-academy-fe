# Multiply by 15

The optimizer never emits a register `mullw` for a constant. For a non-power-of-two like 15, MWCC uses the immediate multiply `mulli r3, r3, 15` — a single instruction.

## Your task
Write `scale` to match the target.

<!-- starter -->
```c
int scale(int x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
int scale(int x) {
    return x * 15;
}
```
