# Logical shift left by 9

`x << 9` is the extended mnemonic **`slwi r3, r3, 9`**, itself a special
`rlwinm` (rotate-left-9, mask the low 9 bits to zero).

## Your task
Write `shl` on a `u32` to match the target.

<!-- starter -->
```c
u32 shl(u32 x) {
    // your code here
    return 0;
}
```

<!-- solution -->
```c
u32 shl(u32 x) {
    return x << 9;
}
```
