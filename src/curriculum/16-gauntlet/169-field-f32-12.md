# Load a `f32` at byte offset 12

A typed field access compiles to a single load with the field's displacement.
A `f32` field reads with **`lfs`** at displacement 12. The *type* picks the load opcode; the *offset* picks the displacement.

```c
typedef struct { u8 _pad[12]; f32 field; } S;
```

## Your task
Write `read` to reproduce the assembly above.

<!-- starter -->
```c
f32 read(S* s) {
    return 0;
}
```

<!-- solution -->
```c
f32 read(S* s) {
    return s->field;
}
```
