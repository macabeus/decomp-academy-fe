# Load a `u16` at byte offset 8

A typed field access compiles to a single load with the field's displacement.
A `u16` field reads with **`lhz`** at displacement 8. The *type* picks the load opcode; the *offset* picks the displacement.

```c
typedef struct { u8 _pad[8]; u16 field; } S;
```

## Your task
Write `read` to reproduce the assembly above.

<!-- starter -->
```c
u16 read(S* s) {
    return 0;
}
```

<!-- solution -->
```c
u16 read(S* s) {
    return s->field;
}
```
