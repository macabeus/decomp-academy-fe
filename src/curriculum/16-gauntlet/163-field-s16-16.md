# Load a `s16` at byte offset 16

A typed field access compiles to a single load with the field's displacement.
A `s16` field reads with **`lha`** at displacement 16. The *type* picks the load opcode; the *offset* picks the displacement.

```c
typedef struct { u8 _pad[16]; s16 field; } S;
```

## Your task
Write `read` to reproduce the assembly above.

<!-- starter -->
```c
s16 read(S* s) {
    return 0;
}
```

<!-- solution -->
```c
s16 read(S* s) {
    return s->field;
}
```
