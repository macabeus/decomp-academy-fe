# Multiply low

PowerPC multiplies two registers with one instruction, **`mullw rD, rA, rB`**
("multiply low word"). It keeps the low 32 bits of the product and discards the
rest:

```asm
mullw r3, r3, r4
blr
```

Notice there's no immediate version of `mullw`. Any time you multiply by a
*variable*, you end up here. Multiplying by a *constant* takes a different path,
which is the next lesson.

## Your task

Write `mul2` so it compiles to the `mullw` above.

<!-- starter -->
```c
int mul2(int a, int b) {
    return 0;
}
```

<!-- solution -->
```c
int mul2(int a, int b) {
    return a * b;
}
```
